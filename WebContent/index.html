Hello AWS DevOps!

Steps
1) Create Simple Web project in Eclipse
2) Create Git project on GitHub Repo
3) Add .gitignore to exclude extra files
4) Get AWS (Register)
5) Get EC2 Instances with PPK (for Jenkins & Test separately)
6) Install requisite softwares on instances
7) Make Master-Slave in Jenkins
8) Configure Jenkins Job and attach shell
9) Configure WebHooks on GitHub and register with Jenkins
10) CD/CI is ready
